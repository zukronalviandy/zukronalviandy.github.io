Instalasi Windows 8

I. Teori Dasar

Windows 8 adalah nama dari salah satu versi Microsoft Windows, serangkaian sistem operasi yang diproduksi oleh Microsoft untuk digunakan pada komputer pribadi, termasuk komputer rumah dan bisnis, laptop, netbook, tablet PC, server, dan PC pusat media.Sistem operasi ini menggunakan mikroprosesor ARM selain mikroprosesor x86 tradisional buatan Intel dan AMD. Antarmuka penggunanya diubah agar mampu digunakan pada peralatan layar sentuh selain mouse dan keyboard, sehingga Windows 8 di desain untuk perangkat tablet sentuh.


Microsoft secara resmi merilis Windows 8 pada 26 Oktober 2012 bersamaan dengan peluncuran komputer perdana yang menggunakan Windows 8.Versi percobaan Windows 8 berupa Consumer Preview dan Release Preview.Pada 26 Juni 2013, Microsoft merilis versi Uji Coba dari Windows 8.1 (sebelumnya dikenal sebagai Windows Blue), sebuah peningkatan untuk Windows 8, Windows 7, Windows Vista, dan Windows XP. Pada 18 Oktober 2013, Microsoft merilis Windows 8.1 sebagai pembaruan gratis untuk Windows 8.


II. Alat dan Bahan

  1. 1 Unit PC
  2. CD Boot Windows 8

III. Langkah Kerja

  1. Masukkan CD Boot Windows 8 kedalam PC

  2. Restart PC

  3. Setting CD Boot Windows 8 menjadi Booting Utama

  4. Tekan tombol sembarang pada keyboard


  5. Tunggu proses loading


  6. Tekan "Next"


  7. Tekan "Install Now"


  8. Masukkan "Product Key"


  9. Centang "Accept the license terms" untuk lanjut ketahap selanjutnya


  10. Pilih "Costum install"


  11. Setting partisi yang ingin diinstalkan Windows 8 atau Partisi C: nya

    Jika ingin menambahkan paritisi tinggal tekan "New"

    Note : Minimal untuk partisi C: adalah 50GB


  12. Jika selesai mensetting partisi-partisi, tinggal menunggu proses instalasi selesai


  13. PC akan otomatis Restart jika proses instalasi selesai


  14. Setting warna kesukaan dan nama PC


  15. Pilih "Express Setting" untuk setting Default


  16. Masukkan Username dan Password

    Password bisa kosong


  17. Windows 8 selesai dan bisa langsung digunakan
