Instalasi Dual OS

I. Teori Dasar

Ubuntu merupakan salah satu distribusi Linux yang berbasis Debian dan didistribusikan sebagai perangkat lunak bebas.


Windows 10 merupakan sistem operasi komputer pribadi yang dikembangkan oleh Microsoft sebagai bagian dari keluarga sistem operasi Windows NT. Diperkenalkan pada tanggal 30 September 2014, dirilis pada 29 Juli 2015 dan pada November 2015, Threshold 2 dari Windows 10 (v10.0.10586) dirilis ke publik.


Untuk penjelasan lebih lanjut tentang Ubuntu dan Windows 10 bisa diklik (Ubuntu) dan (Windows 10)


Pada penginstalan Dual OS ini diperlukan kapasitas Hardisk dan spesifikasi PC yang lumayan besar. Dalam penginstalan ini Main OS-nya adalah Windows 10, karena kita lebih mengerti seluk beluk tentang Windows ketimbang Linux.


II. Alat dan Bahan

  1. 1 Unit PC
  2. CD Boot Ubuntu
  3. CD Boot Windows 10

III. Langkah Kerja

  1. Pertama kita menginstall windows 10 sebagai Main OS (cek disini)

  2. Setelah selesai Windows 10, bagi partisi yang awalnya 2 partisi (D dan C) menjadi 4 partisi di Disk Management pada Windows.
    yaitu : untuk Root(Ubuntu) dan Swap

  3. Reboot PC dengan CD Boot Ubuntu yang telah diload kedalam PC

  4. Instalasi Ubuntu seperti biasa (cek disini)
    Catatan : Partisi yang berisi Windows 10 jangan diformat dan install Ubuntu di kedua partisi yang telah dibuat tadi

  5. Jika semua tahap itu telah dilakukan, berarti PC dengan Dual OS bisa digunakan

V. Kesimpulan

Untuk Dual OS ini sangat berguna untuk orang yang ingin mempelajari tentang Linux lebih jauh. Karena Linux sifatnya Open-Source maka mudah dikonfigurasi.
