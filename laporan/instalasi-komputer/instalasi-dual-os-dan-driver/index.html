Instalasi Dual Windows dan Driver Pack

I. Teori Dasar

Driver pack adalah sebuah software yang menyediakan berbagai Driver untuk Sistem Operasi windows. Driver pack juga menyediakan software-software lainnya seperti Office, Media player, browser dan masih banyak lagi.


Dengan banyak Driver dan Software yang ada didalamnya tidak salah kalau size dari Driver Pack bisa mencapai 16GB dan bisa lebih tergantung kelengkapan Driver dan Software.


II. Alat dan Bahan

  1. 1 Unit PC
  2. CD Boot Windows 8
  3. CD Boot Windows 10
  4. Flashdisk berisi Driver pack

III. Langkah Kerja

  1. Pertama install Windows 10 sebagai Sistem operasi utama dari PC kita, bisa dicek disini
  2. Setelah selesai, bagi partisi yang awalnya 2 menjadi 4 dengan ketentuan partisi untuk Windows 8 lebih dari 50GB
  3. Install Windows 8 kedalam partisi yang 50GB tersebut dengan cara sebagai berikut
  4. Setelah Windows 10 dan Windows 8 dengan sempurna, colok USB Driver Pack kedalam PC
  5. Jalankan Driver Pack
  6. Tekan "Install Automatically" dan tunggu proses hingga selesai
  7. Setelah selesai akan muncul software-software baru yang direkomendasikan diinstall oleh Driver Pack

IV. Hasil

-


V. Kesimpulan

Driver Pack sebenarnya tidak terlalu berguna pada Sistem Operasi Windows 8 dan Windows keluaran baru seterusnya, karena Windows sudah menginstall driver-driver sendiri.
